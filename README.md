一些相关书籍

《人对抗自己：自杀心理研究》

是美国精神分析学家卡尔·门林格尔撰写的心理学著作，1990年首次由贵州人民出版社出版，2004年再版，收录于“现代社会与人名著译丛”系列，由四川大学文学和新闻学院教授冯川翻译。该书基于弗洛伊德的死亡本能理论，系统探讨人类自我毁灭行为的心理动因与社会干预路径。


《能量，性，自杀》

是英国演化生物化学家尼克·莱恩创作的科普著作，由张力翻译，上海译文出版社于2023年6月出版，副标题为“线粒体与生命的意义” 。
全书以线粒体为核心视角，探讨其在生命起源、真核细胞演化、性别分化及衰老机制中的作用。书中提出线粒体内共生起源理论解释真核细胞演化，分析线粒体通过内共生关系驱动能量代谢跃升。
